# Витрина паевых инвестиционных фондов (ПИФ)

Статический сайт-витрина данных о паевых инвестиционных фондах для массового инвестора.
Помогает понять, какой фонд может подойти конкретному пользователю, и сравнить фонды между собой.

Источник данных: Банк России, раздел «Витрина данных» (https://www.cbr.ru/RSCI/data_showcase/), файл `mutual_fund_data.xlsx`, данные актуальны на **30.04.2026**.

Сайт состоит только из HTML, CSS и JavaScript — без сервера, базы данных и API. Может открываться локально или быть размещён на GitHub Pages.

## Структура проекта

```
index.html      — разметка страницы
styles.css       — стили (fintech-дизайн, адаптивная верстка)
script.js        — логика: фильтры, каталог, сравнение, виджет подбора
funds-data.js    — данные фондов в виде JS-массива FUNDS_DATA (генерируется из Excel)
convert.py       — скрипт для конвертации Excel -> funds-data.js
README.md        — эта инструкция
```

## Как открыть сайт локально

Просто откройте `index.html` в браузере (двойной клик или «Открыть с помощью» -> браузер).

Либо запустите локальный сервер (рекомендуется, чтобы избежать ограничений браузера на загрузку файлов):

```bash
cd pif-showcase
python3 -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.

## Как залить проект на GitHub

1. Создайте новый репозиторий на GitHub (например, `pif-showcase`).
2. В папке проекта выполните:

```bash
git init
git add .
git commit -m "Витрина ПИФ: первая версия"
git branch -M main
git remote add origin https://github.com/<ваш-логин>/<имя-репозитория>.git
git push -u origin main
```

## Как включить GitHub Pages

1. Откройте репозиторий на GitHub -> вкладка **Settings** -> раздел **Pages**.
2. В разделе **Build and deployment** выберите источник **Deploy from a branch**.
3. Выберите ветку `main` и папку `/ (root)`.
4. Нажмите **Save**.
5. Через 1-2 минуты сайт будет доступен по адресу:
   `https://<ваш-логин>.github.io/<имя-репозитория>/`

## Как обновить данные из нового Excel-файла

Данные Банка России обновляются ежемесячно. Чтобы обновить витрину:

1. Скачайте новый файл `mutual_fund_data.xlsx` со страницы https://www.cbr.ru/RSCI/data_showcase/
2. Положите файл в папку `Downloads` (или измените путь в `convert.py`).
3. Установите зависимость (если ещё не установлена):

```bash
pip3 install openpyxl
```

4. Запустите конвертацию:

```bash
python3 convert.py
```

Скрипт пересоздаст файл `funds-data.js` с новыми данными и обновит дату актуальности (`FUNDS_UPDATED`).

5. Проверьте сайт локально, затем закоммитьте и запушьте изменения:

```bash
git add funds-data.js
git commit -m "Обновление данных ПИФ"
git push
```

GitHub Pages автоматически обновит сайт через 1-2 минуты после push.

## Важная информация

Материалы сайта носят информационный характер и не являются индивидуальной инвестиционной рекомендацией. Результат работы виджета подбора фонда — ориентир для самостоятельного изучения, а не готовое инвестиционное решение. Перед покупкой паёв необходимо ознакомиться с правилами доверительного управления фондом и информацией на сайте управляющей компании.
